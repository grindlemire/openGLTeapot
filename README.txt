MOVEMENT:
Use arrows to move rotate the teapot around in any direction (note that there is an advanced extremely responsive movement not the standard jerky movement)


MODES:
Y: default mode: Simply the lighting and the vertices with correct normals
T: texture mode: Teapot is textured like it is made out of wood with a correctly themed background
R: Reflection mode: Teapot is textured with an environment reflection map (using sphere map) and is placed in a correct environment that makes it look like it is reflecting what is behind the viewer.
